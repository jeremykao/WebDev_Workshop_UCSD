How the Web Works...

What happens when you type in http://google.com into the URL bar?

Once you connect to a server, what happens?

  • Server runs some code to get what you want.
    • This is called server-side code, collectively called the Backend.
    • Deals with databases, manipulating data, etc.
    • Programming languages include: C++, Java, Node.js, Ruby, PHP, Python, etc.
  • Server sends files to your computer.
  • Your computer downloads the files and displays them.
    • These files are client-side code, collectively called the Frontend.
    • Code for UI, user interaction with web application, etc. All of it is run by your browser!
    • Programming languages include: HTML, CSS, JavaScript

We'll be focusing on the Frontend!

But skills from both frontend and backend overlap

HTML, CSS, JavaScript

Let's use an analogy here. We're gonna build a house.

HTML = materials to build the house (wood, bricks, nails, tables, etc.)

CSS = styling and decorations (paint, placement, etc.)

JavaScript = interactions (properties of objects, etc.)

HTML, CSS, JavaScript

HTML include all elements on a webpage. [Structural]

CSS moves around those elements, colors them, makes them look pretty. [Style]

JavaScript allows users to interact with these elements. [Interactive]
